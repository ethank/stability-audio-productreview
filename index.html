How this works

Run the Monday review from one canonical record instead of a deck.

1

Start the week

Use Start from previous week to pull open commitments into Last Week Follow-up, or Start blank when the team needs a clean slate.

2

Update teams

Select Research, Engineering, or Product, then edit follow-ups, this-week commitments, blockers, and MVPs directly on the board.

3

Use supporting views

Updates captures department-head and team-wide notes. MVPs groups shipped work. History keeps the canonical record visible.

4

Present live

Present mode turns the current record into a shareable walkthrough. No deck export or copy-paste step is needed.

5

Lock the record

Mark ready before review, then lock the week after the meeting so the final state is clear to everyone.

6

Manage access

Admins add users in Settings. Editors can update and lock reviews. Viewers can read and present without changing the record.
